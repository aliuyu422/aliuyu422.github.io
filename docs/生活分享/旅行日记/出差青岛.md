# My Trifle | 青岛



 翻到相册《这座城市》，空间里仍有零零散散的最近访问，那是我记忆里纯净的四年，那些人，好像以后很难再见，所以感觉和现实仿佛隔了一道门帘。



  我开始怀念每每下了飞机坐上机场大巴穿过的那跨海大桥，那段路程是我最期待的，脑海里满是离开了一个寒假暑假后的青岛。对新的学期新的生活又是充满期待。 



  怀念宿舍的他们，上课的时候和卢奇每天踩着点去洗漱上课，即使迟到也不能忘记吃早饭，照烧鸡腿或是麻辣牛肉，在上课钟声响的那一刻消化完荟萃一楼的饼，把剩余的包装纸恰好扔在门口的垃圾桶。



  怀念没课的晚上，在繁星微风的操场看着努力奔跑的模样，或是牵手漫步的情侣，和专业学长学弟们在篮球场上肆意挥洒汗水，在学校北门的烧烤店整上几个扎啤和辣炒蛤蜊。



  这两天被学弟学妹们刷屏，他们现在天南地北地赶回去参加本就属于他们却因去年疫情而亏欠他们的毕业季。



  五月份出差去了济南，在某一个周末萌生了一个突发奇想的念头，索性买了周六去青岛的票。说起来是比较不幸，周六才得知要加班，在高铁上还留下了我加班的影子。中午才到青岛，改签到周日早上七点的高铁。这一算来，好像也只有十几个小时。



  到青岛站，想起的还是那拥挤的隧道公交5678号，每次从学校进城来都是凭借运气挤上去，地铁也如大家所料无休止的延期。但好像这些也并不能阻挡我想仔细端详它的想法。不同的是，在这里我能期待的是好久不见的重逢，一别后再见面，我们的样子。



  娅姐和杰哥是从大学校园一起走向婚姻的一对，去年由于工作太忙没有赶得上他们的婚礼，也是在青协和各种志愿服务中的默契老搭档，现定居在了青岛，过着两人三餐四季的生活。还有老冷，也从隔壁市匆匆赶过来见上一面，以前每周青协开例会都会见面，有时还会请假，但现在大家聚一起见面都甚少了。我想无论如何也得把他们拉回校园，去看看我们的新图书馆和寻找魔性的笑声。



  绕着学校走，到某个地方就能想起以前的事，是那些活动策划到寝室熄灯后回寝的路上，是我们两个专业上一堂毛概课差点打起来的教室，是吃了饭在荟萃广场上坐着看大电视的偶遇。三个食堂，都从一楼逛到了顶楼，看着没太变化的窗口，还能回忆起哪些当年的记忆。



  告别了学校后，想着和辉哥见个面，青岛贵阳来来去去我们好像也都是错过，和辉哥在群里聊得多，但是不怎么太会私信，也还是在说些那些以前的日子和现状，对于未来我们都同样还迷茫，他说下一次不一定是在青岛了，但我还是很期待。到十二点下楼打出租车，走的时候还难得拥抱了下。



  还准备了给湄潭将要毕业的学弟们践行，去了我们平时最爱去的那家口口香餐厅，过了快两年，刚踏进门口，老板娘问我怎么想起回来了是一个人吗，让我们感觉好像随时回来也会有归宿。喝着喝着，后来我们聊了什么我都忘了，无非是在醉酒里感叹无常。



  在青岛的最后个惊喜是毛那两天也在，我半醉半醒地打车回住的地方，见到她后我直接坐在了地上，当时也不知道为什么会下意识有这种反应，等她半搀扶着回了住的地方，本来想说的好多，还说着要去海边浪，都终结在不充裕的旅行时间里。



  青岛的天四点就亮了，我卡着六点的闹铃不耐烦的醒来，路边有跑步的人，也有三两个老人漫步，在很惬意的那条路上打了辆出租车，从后座的窗外闪过的每一帧，哪怕是路墙上的壁虎，都觉得镌美如画。



  路过这一站，还要接着往下走，回见！

# 年度歌手小谈



​        冬日的睡眠很长，抗拒很多，事很繁琐，午休时，想着束手无策的需求辗转反侧，想借一点音乐来转移注意，反而更加烦躁，调低到一丝，低到可以听到同室的酣声，时而也可以听清歌词，时而就飞到昨晚的美梦，就这样过了一个小时，终且无眠。

​        音乐似乎占据了我生活中很大部分的时间，从网络歌手、流行、摇滚、民谣。我自己觉得没有特别钟情的歌手，喜欢也只是一阵儿时间的事，但是过了很久回来听会很容易重新爱上。感慨的也许是那时单曲循环时的环境和心境。

​        网易云2020年的年度喜爱歌手是曾轶可。已经两年蝉联了，从09年快女开始注意她的，像那样的选秀节目我好像追了十年，我只能说谢谢摩登天空，她的词和曲第一眼会让人觉得清新，富有生活气息，又能找到共鸣。

​        QQ音乐2020年的年度喜爱歌手是周杰伦，因为我从小到大并不是听得很多，却是身边朋友们的整个青春，去年KTV偶然一首半岛铁盒走近了耳朵，于是打算开了个会员循环听完了几乎每一首。可是好像我有记忆点的歌曲还是那几首。

​        虾米2020.01.05和我们告别，我才匆匆打开藏在很深地方的它，回忆它，精致的设计，推荐的共鸣，这里好像有很多老歌版权，身边的朋友都在惋惜。世上的告别有千千万，有些事/人在默默陪伴我们，遗憾的是当它离开的时候却才恍然大悟。

​        说来很奇怪，很可能我以后会慢慢成为我眼中父母的那样，他们跟不上我们所谓的流行，他们只会听他们那年代的歌曲，一点儿也不新潮。好像是这样，我也开始觉得很难去接受新的歌。在自己的播放列表一直都是从来听过的歌，去ktv也只能点唱过无数次的歌，反复去循环渗透一些词才能愈发身同感受一点，细细品来愈发觉得有些曲和词真是绝配。

​        我从来都比较偏爱选秀节目里的创作歌手，因为他们以一种特别的方式输出自己的所想，浓缩成一首歌的词。我偶然看到空间日志里天马行空的词，有些青涩，和朋友填的词，在去年还一起唱过。但很真实。最近兴起收购了几张喜欢的CD，未来填满自己的房间。

​        世界很小，记忆有限，总要想记录点自己的足迹，才能让自己待得更久一点。希望你能在另一个安静的世界找到自己，一哆嗦，吓得看了看旁边爬满了灰的吉他。

​        很烦周日的晚上，像是所有美好都把我抛弃，让我孑然去面对所有痛苦。打喷嚏和整理东西一样，完成的那一刻顿时神清气爽。

​        最近单曲循环听的歌手是张悬。最近爱看剧的是【你好生活】，最近枕边的书是民法典。

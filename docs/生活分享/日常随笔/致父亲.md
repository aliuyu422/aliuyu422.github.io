# My Trifle | 父亲



从小到大，父亲把我照顾得很好。

照片中的他一副黄色眼镜，年轻的时候就拥有中年“油腻大叔”的肚腩，竟还随身抱着一个吉他。

那个黄色眼镜戴了二十几年，肚腩也一直端着，吉他还在书柜的顶层生了灰。

据说，他选择买下了一辆和县城门面一样贵的摩托车，为了家庭每天往返跑从黄家坝到湄潭接送妈妈，我被妈妈背起来，一起坐在摩托车上，狭窄的车上好像是我们三个人靠得最近的时候。有一次我在车上从背带掉下去了，他们走了好久才发现我掉了。如果没发现或者晚一点，可能现在网上拼命寻亲也该有我的身影了。

三岁时我们搬到了湄潭，他每天还是要往返跑很长的路程，记得小时候我不让他走，妈妈会用其他东西或者事物吸引我的注意力，他偷偷地溜走，走了好一会儿然后妈妈给我唱“你爱的人已经飞走了”。我就去窗台上目送他离开的身影，有时他也会回头给我挥手，他总把我举得很高，记忆中他的胡渣总是很刺人。

上了小学到三年级，慢慢觉得他是一个全能的超人。学校经常会有一些绘画比赛和手抄报比赛，有一次语文老师必须让我们每个人办一份手抄报，确实也难倒从小就很听老师话的我，但是碍于也并没有遗传到这个天赋，还是他，妈妈说在深夜一点过帮我完成了作业。当然，第二天这份手抄报也当着全班同学的面得到了褒奖，可是大家都知道这并不是我自己完成的，这份手抄报在学校的专栏里贴了很久。但是我自己也许久没有过得了自己这关，以后有类似的比赛，老师还是会首先让我积极参与，只是后来他教导我不会的东西慢慢可以学，直到我后来可以单独完成绘画和手抄报。他教会了我好多，游泳、骑单车、打球、放风筝，也给了我很多鼓励。夏天的晚上，我会问他这个世界如果毁灭了，我们怎么办之类的问题。外面的蝉鸣也伴随鼾声睡去，第二天他早早地就去上课了。

再长大一点，我和他开始有了争执，我开始不理解他的做法，我甚至觉得他有时候触及到我的自尊心和虚荣心，关于那时候住的房子，隔音效果不好，每次吵架都会让街坊邻居看够了笑话。妈妈每天给我俩做思想工作，也好像每次也都是他来给我道歉。

上了初中，他陪着我做数学题竞赛，给我兴趣给我讲题思路，教我物理教我化学，提醒我每次考好时不要骄傲，考差时就带着我去姑妈家里闭关收心。我好像比一般人更早一点理解父母，以至于我妈还经常和她的姐妹们说我的叛逆期还没到。从那次汶川大地震后，我们很少吵过架了。

上了高中，我就住在我外婆那儿，他们会每周三都来看我一次，高三最后几个月也是每天都会来接我回家，而且我还记得每次出校门总能第一眼看到他对我笑的样子。

去了大学，我基本上很少能和他有语言了，我俩之前好像谈心次数屈指可数，他知道我是一个不会乱花钱的人，我俩每次聊天不超过十句就会戛然而止，我们都只是想真的关心一下彼此，但是我们好像从来不善于表达出来。

后来在贵阳工作，爸爸经常会问我累不累，还在加班吗，习惯了吗，我不再想让他们担心了，每次我总是把最近的开心事都告诉他，内心好多匹草泥马都憋在心里，我想让他们知道我过得可以， 不要为我担心。

***他是个什么样的人***

印象中他从来没有打过我，无论我做得多么过分的事情，他也是极力控制自己的情绪出门自己冷静。

他看上去严肃，我表姐表弟虽然不说但是都很怕他，我同学也很怕他，因为他不经常笑且时常摆着一副不那么好看的嘴脸。

他很节俭，他舍不得给自己买衣服和鞋，他舍不得很多丢掉的东西，甚至连我爱吃老干妈，都给我在从中偷偷添油加醋。

他身体不好，需要长期服药，还好在早期检查出来了，导致不能运动，只能控制饮食，他能把陪他很久的肚腩一直保持。

他生气很容易看出来，他生气是把舌头放在外面，但是慢慢我很少看到他这样，不知道是他把委屈都藏得更深了还是他不会和我表达气愤了。 

他朋友太少了，他和我说他有时候很高兴我能够交到很多好朋友，他不打牌不抽烟不喝酒，以前和我说的那个最好的朋友似乎也没有理由因为爱好一起聚了。

他有一个当兵梦，大学军训的时候我和他说我今天握到真枪了，很尴尬三次都脱靶了，他很自豪地说他之前打枪都是接近靶心。

他是个大学霸，初中读完后直接读了中专当老师，那时候算得上县里前几名，奶奶说家里面没有钱，他是老大哥，他经常背着我幺爸打着煤油灯学习。

他是个垃圾桶，等我们吃完了，他才会把自己的饭量表现出来，什么都舍不得扔掉，往自己嘴里送，现在也没改过来。有一次破天荒带我去吃牛排，去了才知道他看到招牌写的周三半价。

他是个全能选手，电冰箱洗衣机电视各种电器他都能修，也能给我补裤洞能去商店砍价。

他也能下得厨房，有一次我过生日想请同学去餐馆吃顿饭，但是他第二天忙了一天邀请我的朋友们来，给我们做了满满的两桌菜后自己出去了。

他不知道我喜欢吃什么，但是让他知道了就会给我买很多很多，比如批发方便面、葡萄、冰棍。

他是家里的主心骨，从爷爷奶奶到他的六个弟弟，遇到大事都是他全面分析，敲定主意，大事小事都要经过他的意见，似乎家里的大事都会听从他的决定性意见。

他是个冷酷的人，我基本上没看到他哭，除了上次过年闹翻了和他谈了一次心，每次都会说：“哭什么，有什么好哭的”。

他爱撒谎，刚刚端上来的肉他说他不爱吃，他说他身体挺好的，他说他不需要礼物，不在意我去不去看他只要我回湄潭自己玩得开心就好。

他爱炫耀，在姑姑给我看的朋友圈里发过我的荣誉证书，我书写端正的英语作文，把我的照片一直当成他的屏保。

虽然很遗憾他和妈妈分开了，回去之后见到他的次数更少了，记忆中和他只谈过一次心，给他写过一次信。我害怕也许哪天没有机会了，再多的感情都不会表达出来，但是我会用文字记录下来，用心记住他给我的，说不定哪天再发给他，再用我慢慢的努力让他们知道我真的不是小孩子了。

他做我的后盾真的很称职。让我在任何崩溃的时候看到以上的文字都有一种力量去面对。因为我也有一天也会成为父亲，我能向他学习的还有太多太多。

我只希望他永远健康，能够等待我有更多能力报答他的时候。

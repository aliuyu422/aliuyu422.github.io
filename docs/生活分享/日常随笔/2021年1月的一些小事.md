# My Trifle | 答案



 我记得小时候就说我随她的犟脾气。一旦自己的满怀期待正巧撞到了迟疑，我们都宁愿不要。



​    总是说要过得洒脱，可静下来的时候，我们该想什么，我还是不知道拿什么去分神。最好的朋友说是我是他见过男生里最细腻的人，可这个词好像不可逆，一旦进去，好像无法退回。



​    我所感觉到，这个世界一直都对我进行降维打击。我看到的感动、幸福、喜悦似乎都好像是最表面的一层，失望、无助、自我怀疑都是最灵魂的质问。



​    长大后，任何你曾经引以为傲的事都会有人完成过，任何的褒奖都并非自己独一无二，脱口而出的夸奖都是别人的尊重和鼓励罢了，慢慢学着去接受平凡，接受规则。



​    若有所思的旁人看着我，像看着在摇篮里因为风吹起的纸风车而手舞的我。我觉得自己好渺小，渺小到看到的事物都那样浅薄，感受到的都是单纯的喜怒哀乐，且大悲大喜大哀大乐。



​    我有时在意我的朋友怎么看我，我曾经问过朋友一个问题，你希望你的朋友怎么形容你。这个问题不在于你在不在乎别人怎么看待你，而是在于在你亲近的人面前你向这个世界输出了哪些特质。



​    我想了很久，至于真的太多了，那就是一个词完美。我实际上也是在努力，我好像不太懂得如何拒绝，我好像尽力地想帮助到每个人，想努力的维持任何一个像要走远的朋友。我有时候是个垃圾场，在朋友安静的倾听他们的苦诉，我像是个驿站，我在认真倾听完后最真实的表达自己的建议，我像是个避难所，最低落的时候我常常也陪到聚会的最后。可是有些该走的还是要走，比如朋友、感情、时间。 



​    老说别患得患失，可我忍不住去想了想自己的收益。我和朋友距离暂时更近了，或者他难忘这一次的醉酒是我搀扶着到家，我曾是他这段时间最好的倾诉对象。这些啊，迟早会被代替，会被遗忘。人的一生，很多东西都会收敛于一个值，在时间轴越拉越远，那些富余的朋友总会慢慢走丢，富余的爱好会慢慢摒弃，富余的圈子会悄悄消失。



​    你还是在去认真计算真心的关系能够撑多久，你多累啊。也许你在别人看来真的做得没有那么好，也许也都厌倦了你的方式，也许某个细节早都让人想疏远。可是你还是在自我感动些什么。人生苦短，做些让自己安心和有意义的事吧。



​    做了个梦，没有痛苦，甚至很甜。等明天太阳爬上来的时候，醒来不就好了吗。会有更好的吗，会的，就像我也在变得更好不是吗。
